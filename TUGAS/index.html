Pengertian Tag

Tag adalah sebuah penanda awalan dan akhiran dari sebuah elemen di HTML. Tag dibuat dengan kurung siku (<...>). Lalu di dalamnya berisi nama tag dan kadang juga ditambahkan dengan atribut.

Contoh tag HTML :

  • < html > : untuk memulai html
  • < head > : untuk bagian kepala dokumen
  • < body > : untuk bagian tubuh dokumen
  • < p > : untuk membuat paragraph

Atribut HTML

Atribut adalah kata kunci khusus yang berada di dalam tag pembuka. Atribut juga disebut sebagai modifier yang akan menentukan perilaku dari elemen.

Atribut dapat ditambahkan pada elemen manapun. Ada juga elemen yang mewajibkan menggunakan atribut seperti elemen < a >,< img >,< video >, dll.

Nilai Value Atribut

Dalam konteks HTML, nilai (value) dalam sebuah atribut adalah informasi tambahan yang diberikan untuk menentukan karakteristik atau properti dari elemen HTML tersebut. Atribut terdiri dari dua bagian : nama (name) dan nilai (value). Nilai ini biasanya ditulis didalam tanda kutip setelah nama atribut.

Contoh sederhana :

Diatas, src adalah nama atribut yang menentukan sumber gambar, dan "gambar.jpg" adalah nilai atribut tersebut. Begitu juga alt adalah nama atribut yang memberikan teks alternatif untuk gambar, dan "Deskripsi Gambar" adalah nilai atributnya

Struktur HTML

Pada HTML terdapat kerangka inti yang membangunya seperti tag, element, dan attribut.

  1. Tag

    2. Tag merupakan bagian pembuka dan penutup yang digambarkan dengan <..> lalu dalamnya akan diisi dengan nama tag dan atribut yang nantinya akan diproses oleh browser. Setiap tag memiliki fungsinya masing-masing. Misalnya kamu ingin membuat judul maka menggunakan tag < h1 >< /h1 > dan jika ingin menambahkan image maka menggunakan tag < img >.

  2. Element

    3. Element adalah keseluruhan dari awal tag hingga penutup tag. Berarti element berisi tag pembuka, isi tag, dan tag penutup. Contohnya seperti ini. < h1 >Belajar HTML Bersama DomaiNesia < /h1 >.

  3. Atribut

    4. Attribute adalah informasi yang bisa ditambahkan dalam setiap element untuk mengatur dan menambahkan sesuatu. Misalnya dalam penggunakan tag images. Maka dapat dilengkapi menjadi < img src=”html.jpg” alt=”html”/ >. Dari tag tersebut artinya bahwa image akan memproses gambar komputer dan alt akan menampilkan tulisan “html” jika gambar gagal diproses.
